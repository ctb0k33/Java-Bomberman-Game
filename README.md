# Java-Bomberman-Game
Java Bomberman Game built with FXGL 

## Contributors
* Trần Chiến Thắng. <br/>
* Mạc Gia Khánh. <br/>
* Phạm Minh Hiếu.

## Introduce
This game is built base on Bomberman game. <br />
The aim of the game is destroy all the enemies and portals that exist on the map.

## Entity
Baromu: <br/>
This eneny moves randomly. <br />
